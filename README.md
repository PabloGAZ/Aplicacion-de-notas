Backend: NodeJS + Express

DB: mongoDB



<br>

# Cómo iniciarlo 

-Descargar el repositorio desde Github

-Sitúate desde la terminal en la carpeta del proyecto y ejecuta el siguiente comando: 
npm run dev

El proyecto se abrirá en localhost:3000
